Inventory Manager App

GitHub

A straightforward inventory manager app in Android. Track specific attributes, built-in barcode scanning, and more.

Android Studio Java Firebase

How does it work?

This is a Java Android app built on Firebase. Built in a complete Agile lifecycle from user stores to unit test coverage, users can add items to the Inventory Manager with fundamental attributes like date acquired, and price, as well as more specific attributes. Items can be quickly grouped and sorted using the filtering, sorting, and tagging system.

Features:

  • Advanced item filtering and sorting
  • Barcode Scanning and built-in item lookup
  • Item tagging
